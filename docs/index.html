Download and import

You can download Java Security Essentials here. Then import it in your project as a Java library (.jar).

Hashing

Java Security Essentials allows you to obtain a hash representation of data using either the default algorithm or other available hashing algorithms.

You can use the default hashing algorithm (SHA512) to hash an array of bytes using the Hashing.hash() method.
Note that you have to convert your data (text, numbers, objects etc) into an array of bytes before hashing it:

final String input = "MyRandomInput";
String hash = Hashing.hash(input.getBytes());
Hashing algorithms

You can also use other available hashing algorithms listed below (HashType), using the same function but providing the algorithm first:

  • MD5
  • SHA1
  • SHA256
  • SHA384
  • SHA512 (default)
String hash = Hashing.hash(HashType.MD5, input.getBytes());

It is suggested to use the SHA512 (the default algorithm) when hashing as it is more secure.

Hashing with salt

To make your hashes more secure, you need to use salt. Salt provides randomness to the algorithms, making the resulting hash computationally infeasible to guess or find using a rainbow table. A salt is a random string of bytes that randomizes the output of the algorithm. To use salt, you need to create one and store it securely so that you can use it to re-create the same hash for the same data again if needed. To create the salt use the Hashing.salt() method: 

final byte[] salt = Hashing.salt();

Then, you can pass the salt (pun intended) as a parameter to the hash() method:

Hashing.hash(input.getBytes(), salt);
//OR
Hashing.hash(HashType.SHA512, input.getBytes(), salt);

Remember to store the salt in a file or database, otherwise you will not be able to re-create the same hash.

Symmetric encryption

You can symmetrically encrypt text with a key/password using the SymmetricEncryption class. Under the hood, this class utilizes the Advanced Encryption Standard (AES) algorithm, which is the adopted worldwide standard for symmetric encryption. To encrypt a text, use the SymmetricEncryption.encrypt() method with your input and key/password as parameters:
